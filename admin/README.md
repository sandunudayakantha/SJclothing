# SJ Clothing Admin Panel

React + Vite + Tailwind CSS admin panel for managing the store.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (optional):
```env
VITE_API_URL=http://localhost:5007
```

3. Start development server:
```bash
npm run dev
```

The admin panel will run on `http://localhost:5174`

## Login

Use the admin credentials created via the register endpoint (see main README).

## Features

- Dashboard with statistics
- Product management
- Category management
- Order management
- Store settings

## Build

```bash
npm run build
```
