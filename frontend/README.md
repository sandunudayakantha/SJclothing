# SJ Clothing Frontend

React + Vite + Tailwind CSS frontend for the main store.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (optional):
```env
VITE_API_URL=http://localhost:5007
```

3. Start development server:
```bash
npm run dev
```

The app will run on `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- Home page with hero, featured products, new arrivals
- Product listing and detail pages
- Shopping cart and wishlist
- Checkout process
- Contact page
